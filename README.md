# LLM-Guided-Latent-Evolutionary-Optimization-for-Molecular-Property-Targeting

Molecular property optimization code for LLM-guided latent evolutionary search.

This repository contains code for molecular property targeting experiments using
PS-VAE latent representations, latent-space genetic algorithms, LLM-guided seed
generation, hybrid initialization, and logP transfer experiments.

## Contents

- `Main_results_202604_LLM_GA/`: main QM9 target-property experiments.
- `Ablation_1/`: seed strategy and iterative guidance ablation experiments.
- `Gen_Exp/`: logP transfer/generation experiments and PS-VAE audit utilities.
- `QM9_test/PS-VAE/`: PS-VAE model code and related training/evaluation scripts.

Large datasets, trained checkpoints, generated result folders, logs, API keys,
and local model caches are intentionally excluded from version control.
